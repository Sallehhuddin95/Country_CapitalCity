# Country Capital Name

This repository lists out all the country names and their corresponding capital cities. It serves as a reference for developers and individuals who need to access this information programmatically or for general knowledge purposes.

Feel free to explore the list and use it in your projects or applications.

## Usage

To use this repository, simply navigate through the files and find the country you are interested in. Each country is listed alphabetically, making it easy to locate the desired information.

## Contribution

If you notice any missing or incorrect information, please feel free to contribute by submitting a pull request. Your contributions are greatly appreciated and will help keep the repository up-to-date.
